## 🥇 유형 1: 변의 비 → 넓이 비 (9문제, 47%) ← **진짜 약점**

### 문제 패턴
- 삼각형/사각형 안의 점들이 변을 어떤 비로 나눔
- 그 점들로 만든 새 도형의 넓이를 구함

### 해당 문제 (9개)
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/2_4` | 직사각형 4분할 (65, 35, 49 m²) |
| `31. Mock/2_26` | △DEF=7로 △ABC 넓이 |
| `31. Mock/1_26` | 삼각형 ABC 내부 점들로 △EFG |
| `31. Mock/4_15` | 사다리꼴 ABCD 음영 넓이 |
| `31. Mock/4_21` | 정육각형 각 변 2배 연장 → 큰 육각형 |
| `02. NM 2010/25` | 삼각형 BEF 넓이로 사각형 AEFC |
| `53. RI 2023/18` | 질량점 활용 (BD=DE=EC, CF:AC=1:3) |
| `54. RI 2024/11` | BD=3/7 BC, AF=FD 음영 |
| `55. RMO 2025/03` | 정사각형 내부 마름모 넓이비 |

### 필요한 핵심 무기 3개

#### ① "같은 높이면 밑변 비 = 넓이 비" (기초)
- 한 직선 위 두 삼각형은 높이가 같음 → 넓이비 = 밑변비
- **예**: BD : DC = 2 : 3 → △ABD : △ACD = 2 : 3

#### ② "한 꼭짓점 공유 → 두 변의 비 곱" (핵심)
- △ABC에서 D가 AB 위 (AD:DB = m:1), E가 AC 위 (AE:EC = n:1)
- △ADE : △ABC = (m·n) / ((m+1)(n+1))
- **이 공식 자동화 안 되면 위 문제들 다 막힘**

#### ③ "닮음비² = 넓이비"
- 닮은 도형이면 변의 비를 제곱하면 넓이의 비
- 예: 변 비 2:3 → 넓이 비 4:9

### 학습 순서 (3주)
- **1주차**: ① 매일 5문제 (밑변비=넓이비) → 자동화될 때까지
- **2주차**: ② 한 꼭짓점 공유 공식 외우고 매일 3문제
- **3주차**: 위 9문제 다시 풀기 → 같은 패턴 인식 훈련

### 추천 자료
- 중2 닮음 단원 + 디딤돌/쎈 수학 응용 단계
- 영재고 대비 **"Routh 정리"** 검색 (위 ② 공식의 일반화 형태)

## 🥈 유형 2: 양 이동/변화 후 비율 (5문제, 26%)

### 문제 패턴
- 처음 두 양 → 일부 이동 (옮김/추가/제거) → 새 비율
- 처음 양 또는 이동량 구하기

### 해당 문제 (5개)
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/2_25` | 두 통 기름 P→Q→P 이동, 최종 11:7 |
| `31. Mock/4_6` | 빨간/흰 공 비율 변화, 추가량 차이 64 |
| `31. Mock/4_28` | 가격 변동 후 새로운 질량비 m:n |
| `53. RI 2023/12` | B마을→A마을 이동 후 남자 비율 변화 |
| `54. RI 2024/14` | 구슬 이동 후 비율 조건 2개 |

### 필요한 무기

#### ① "불변량을 찾아라" ⭐
- 두 양 **합 = 일정** (단순 이동만 있을 때)
- 한 종류의 양 = 일정 (다른 종류만 추가/제거 시)
- **이걸 먼저 찾으면 식 단순화**

#### ② "비율 = m/n으로 두고 식 세우기"
- 처음 = m, n으로 두기
- 변화 후 = 새 비율 → 비례식
- 외항·내항 곱 = 같다

### 학습 팁
- "옮기기 전 → 옮긴 후" **표 그리기 습관** 필수
- 5문제 풀어보면 패턴이 보임

## 🥉 유형 3: 연비 (비율 합치기) (2문제, 11%)

### 문제 패턴
- A:B, B:C 두 비를 합쳐서 A:B:C로
- 또는 여러 그룹의 비를 통합

### 해당 문제 (2개)
| 파일 | 한줄 요약 |
|---|---|
| `53. RI 2023/07` | 수입 4:3, 지출 11:7, 저축 같을 때 월 수입 |
| `60. AP 2025/10` | S1:S2:S3:S4 비율 합치고 총원 2158명 |

### 필요한 무기

#### ① 공통항 통일
- A:B = 2:3, B:C = 4:5 → B를 12로 통일 → **A:B:C = 8:12:15**

#### ② 가비의 리 (Garbi's Theorem)
- a:b = c:d 이면 (a+c) : (b+d) = a:b
- 합/차 조건과 비율을 묶을 때 강력

### 학습 팁
- 4~5문제만 풀면 패턴 익숙해짐 (간단한 영역)
