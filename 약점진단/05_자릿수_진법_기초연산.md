## ① 자릿수 변수화 & 진법 (10문제)

### 문제 패턴
n자리 수의 각 자리를 변수로 두고 조건을 식으로.

### 핵심 무기
- **3자리**: abc = 100a+10b+c
- **자리 바꿈**: abc ↔ cba 차이 = 99(c-a)
- **십·일 자리 바꿈**: abc → acb 차이 = 9(c-b)
- **n진법 ↔ 10진법** 익숙해지기

### 해당 문제

|---|---|
| `31. Mock/1_12.png` | 2(abcd)+1000=dcba 만족 네 자리수 |

| `31. Mock/2_31.png` | 6개 세자리 등차수열 합 최솟값 |
| `31. Mock/5_25.png` | 4자리 수의 첫 자리를 끝으로 옮기면 4707 커지는 최소 수 |
| `51. RI 2021/14.png` | 11부터 시작 팰린드롬 나열에서 661번째 자릿수 |
| `55. RMO 2025/12.png` | 3자리 수에서 십의 자리가 나머지 두 자리 평균 |

## ② 9의 배수·11의 배수 판정 (3문제)

### 문제 패턴
자릿수 합 ⇔ 원래 수의 mod 9 / 11.

### 핵심 무기
- **9의 배수**: 자릿수 합이 9의 배수
- **11의 배수**: 홀수번째 자리 합 - 짝수번째 자리 합 = 11의 배수
- **미지 자리 찾기**: 큰 수의 미지수 위치를 mod 조건으로

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `02. NM 2010/29.png` | 2010년 날짜 자릿수 합이 12인 일수 |
| `31. Mock/2_19.png` | 1~9 한번씩 써서 만든 그룹 내 소수 최대 |
| `31. Mock/2_35.png` | 자릿수 합 행 정렬에서 9799 위치 |

💡 **Tip:** 큰 자릿수 문제는 항상 '9의 배수 / 11의 배수' 먼저 시도.

## ③ 자릿수 패턴 (이어쓴 수) (2문제)

### 문제 패턴
1, 2, ..., N을 이어쓴 큰 수의 자릿수 분석.

### 핵심 무기
- **1~9**: 9자리, **10~99**: 180자리, **100~999**: 2700자리
- **N번째 자리 찾기**: 누적으로 어느 수의 어느 자리인지

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/5_35.png` | 1~100 이어 쓴 후 3자리씩 묶었을 때 최대·최소 차이 |
| `55. RMO 2025/01.png` | n번째 정수가 n번 반복되는 수열의 2025번째 자릿수 |

💡 **Tip:** 이어쓴 수 문제는 자릿수 구간별로 나눠 누적.

## ④ 기초 연산 (분수·소수·단위·요일) (8문제)

### 문제 패턴
분수/소수의 사칙연산, 단위 환산, 요일 추적.

### 핵심 무기
- **분수 계산**: 통분, 약분 자동화. 가분수↔대분수
- **소수 계산**: 자리 맞춰 더하기/빼기
- **단위 환산**: 시·분·초, m·cm 등
- **요일 계산**: mod 7 활용 (1주 = 7일)

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/2_12.png` | 1/(1/2016+..+1/2011) 정수부분 |
| `31. Mock/5_8.png` | Sandra·Hans·Joe 480달러 분배에서 처음 Hans의 금액 |
| `51. RI 2021/11.png` | 1/Σ43, 1/Σ(43+86)… 합의 값 |
| `53. RI 2023/01.png` | Σ1/[k(k+1)] > 1823/2023 만족 최소 n |
| `54. RI 2024/05.png` | 1/(2²-1)+...+1/(2024²-1) 의 분자·분모 자릿수 합 |
| `54. RI 2024/10.png` | 이중합을 재배치해 정수합 형태로 계산 |
| `01. NM 2008/02.png` | 12시간/일 작동 시 $864000 인쇄 일수 구하기 |
| `60. AP 2025/04.png` | 2024년 2월 29일이 목요일 → 다음 목요 윤년 |

💡 **Tip:** 분수 계산은 매일 5문제 꾸준히. 자동화가 답.
