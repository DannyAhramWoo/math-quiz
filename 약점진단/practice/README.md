## 📝 사용 방법

1. **단원 선택**: 위 표에서 약한 단원의 MD 파일 클릭
2. **세부개념별로 5문제씩** 묶여 있음 → 한 개념씩 연습
3. **답·풀이가 바로 아래** 있음 → 먼저 가려두고 풀기, 나중에 확인
4. **모두 맞으면** 그 개념은 자동화 OK → 다음 개념으로
5. **2개 이상 틀리면** → 해당 [핀포인트 분석 MD](../README.md)에서 핵심 무기 다시 보기
