# 방정식 · 식조작 · 인수분해 — 개념 연습 문제

## 세부개념 1: 방정식 (연립 포함)

**문제 1.** 다음 일차방정식을 풀어라: 3x - 5 = 7.

답: **x = 4**
풀이: 양변에 5 더하면 3x = 12. 양변을 3으로 나누면 x = 4.

---

**문제 2.** 다음 방정식을 풀어라: 2(x + 3) = 14.

답: **x = 4**
풀이: 분배법칙으로 2x + 6 = 14. 양변에서 6 빼면 2x = 8. x = 4.

---

**문제 3.** 다음 연립방정식을 풀어라: x + y = 10, x - y = 4.

답: **x = 7, y = 3**
풀이: 두 식을 더하면 2x = 14, x = 7. y = 10 - 7 = 3.

---

**문제 4.** 다음 방정식을 풀어라: 5x - 3 = 2x + 9.

답: **x = 4**
풀이: 양변에서 2x 빼면 3x - 3 = 9. 양변에 3 더하면 3x = 12. x = 4.

---

**문제 5.** 다음 연립방정식을 풀어라: 2x + y = 11, x + y = 7.

답: **x = 4, y = 3**
풀이: 첫째 식에서 둘째 식을 빼면 x = 4. y = 7 - 4 = 3.

---

## 세부개념 2: 식 인수분해

**문제 1.** 다음 식을 인수분해하라: 3x + 6.

답: **3(x + 2)**
풀이: 공통인수 3을 묶어내면 3(x + 2).

---

**문제 2.** 다음 식을 인수분해하라: x² + 5x.

답: **x(x + 5)**
풀이: 공통인수 x를 묶어내면 x(x + 5).

---

**문제 3.** 다음 식을 인수분해하라: 4a² - 8ab.

답: **4a(a - 2b)**
풀이: 공통인수 4a를 묶어내면 4a(a - 2b).

---

**문제 4.** 다음 식을 인수분해하라: 6x² + 9x.

답: **3x(2x + 3)**
풀이: 공통인수 3x를 묶어내면 3x(2x + 3).

---

**문제 5.** 다음 식을 인수분해하라: 2xy + 4y.

답: **2y(x + 2)**
풀이: 공통인수 2y를 묶어내면 2y(x + 2).

---

## 세부개념 3: 제곱차 인수분해

**문제 1.** 다음 식을 인수분해하라: x² - 9.

답: **(x + 3)(x - 3)**
풀이: 제곱차 공식 a² - b² = (a+b)(a-b)에서 a=x, b=3.

---

**문제 2.** 다음 식을 인수분해하라: x² - 25.

답: **(x + 5)(x - 5)**
풀이: 25 = 5². (x + 5)(x - 5).

---

**문제 3.** 다음 식을 인수분해하라: 4x² - 1.

답: **(2x + 1)(2x - 1)**
풀이: 4x² = (2x)², 1 = 1². (2x + 1)(2x - 1).

---

**문제 4.** 제곱차 공식을 이용해 계산하라: 102 × 98.

답: **9996**
풀이: 102 × 98 = (100 + 2)(100 - 2) = 100² - 2² = 10000 - 4 = 9996.

---

**문제 5.** 다음 식을 인수분해하라: 9a² - 16b².

답: **(3a + 4b)(3a - 4b)**
풀이: 9a² = (3a)², 16b² = (4b)². (3a + 4b)(3a - 4b).

---

## 세부개념 4: 분배법칙

**문제 1.** 분배법칙으로 전개하라: 3(x + 4).

답: **3x + 12**
풀이: 3 × x + 3 × 4 = 3x + 12.

---

**문제 2.** 분배법칙으로 전개하라: 5(2a - 3).

답: **10a - 15**
풀이: 5 × 2a + 5 × (-3) = 10a - 15.

---

**문제 3.** 분배법칙으로 계산하라: 7 × 99.

답: **693**
풀이: 7 × 99 = 7 × (100 - 1) = 700 - 7 = 693.

---

**문제 4.** 분배법칙으로 전개하라: (x + 2)(x + 3).

답: **x² + 5x + 6**
풀이: x·x + x·3 + 2·x + 2·3 = x² + 3x + 2x + 6 = x² + 5x + 6.

---

**문제 5.** 분배법칙으로 계산하라: 25 × 16 + 25 × 4.

답: **500**
풀이: 25 × (16 + 4) = 25 × 20 = 500.

---

## 세부개념 5: 식 조작

**문제 1.** x + y = 7, xy = 12일 때 x² + y² 의 값은?

답: **25**
풀이: x² + y² = (x + y)² - 2xy = 49 - 24 = 25.

---

**문제 2.** a - b = 3, ab = 10일 때 a² + b² 의 값은?

답: **29**
풀이: a² + b² = (a - b)² + 2ab = 9 + 20 = 29.

---

**문제 3.** x + 1/x = 4일 때 x² + 1/x² 의 값은?

답: **14**
풀이: x² + 1/x² = (x + 1/x)² - 2 = 16 - 2 = 14.

---

**문제 4.** x + y = 5, x² + y² = 13일 때 xy의 값은?

답: **6**
풀이: (x + y)² = x² + 2xy + y²에서 25 = 13 + 2xy. 2xy = 12, xy = 6.

---

**문제 5.** a² + b² = 20, ab = 8일 때 (a + b)² 의 값은?

답: **36**
풀이: (a + b)² = a² + 2ab + b² = 20 + 16 = 36.

---

## 세부개념 6: 선형결합

**문제 1.** 3x + 2y = 12, x + y = 5일 때 x + 0·y 의 값은?

답: **x = 2**
풀이: 둘째 식에 2 곱하면 2x + 2y = 10. 첫째에서 빼면 x = 2.

---

**문제 2.** 2x + 3y = 13, 2x + y = 7일 때 y의 값은?

답: **y = 3**
풀이: 첫째에서 둘째 빼면 2y = 6, y = 3.

---

**문제 3.** a + 2b = 10, a + b = 7일 때 a, b의 값은?

답: **a = 4, b = 3**
풀이: 첫째에서 둘째 빼면 b = 3. a = 7 - 3 = 4.

---

**문제 4.** 4x + y = 14, x + y = 5일 때 x의 값은?

답: **x = 3**
풀이: 첫째에서 둘째 빼면 3x = 9, x = 3.

---

**문제 5.** 3a + 2b = 16, a + 2b = 10일 때 a의 값은?

답: **a = 3**
풀이: 첫째에서 둘째 빼면 2a = 6, a = 3.

## 세부개념 7: 최적화

**문제 1.** 합이 10인 두 양수의 곱이 최대일 때 곱의 값은?

답: **25**
풀이: 두 수가 같을 때 (각 5) 곱이 최대. 5 × 5 = 25.

---

**문제 2.** 합이 12인 두 양수의 곱이 최대일 때 두 수는?

답: **6과 6**
풀이: 합이 일정할 때 두 수가 같으면 곱 최대. 12 ÷ 2 = 6.

---

**문제 3.** 둘레가 20인 직사각형의 넓이가 최대일 때 넓이는?

답: **25**
풀이: 가로 + 세로 = 10. 가로 = 세로 = 5일 때 최대. 5 × 5 = 25.

---

**문제 4.** 곱이 36인 두 양수의 합이 최소일 때 두 수는?

답: **6과 6**
풀이: 곱이 일정할 때 두 수가 같으면 합 최소. √36 = 6.

---

**문제 5.** 둘레가 24인 직사각형의 넓이의 최댓값은?

답: **36**
풀이: 가로 + 세로 = 12. 가로 = 세로 = 6일 때 최대. 6 × 6 = 36.
