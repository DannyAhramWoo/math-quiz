## 🎯 사용 방법

각 파일은 **무기 단위**로 묶여 있음 (예: "외각 정리", "별과 막대 공식", "포함-배제 공식"):

```
## ① 외각 정리
> **무기**: 삼각형 외각 = 인접하지 않은 두 내각의 합

문제 1, 2, 3 + 답 + 풀이
```

### 추천 학습 순서

1. **개념 자동화 단계** (이 폴더, 무기 연습)
   - 무기 하나씩 3문제 모두 맞으면 다음으로
   - 1번이라도 틀리면 → 그 무기에 익숙해질 때까지 비슷한 문제 더 풀기
2. **개념 연습 단계** (`../` 폴더, 일반 개념 연습)
   - 무기들을 종합한 5문제씩
3. **응용 단계** (`../../` 약점진단 폴더, 핀포인트 분석)
   - 실제 경시 문제 다시 풀기

## 💡 효율 팁

- 무기 = **도구**. 하나하나 자동화 (생각 없이 쓸 수 있을 정도)
- 매일 한 단원의 무기 3-5개씩 → **2-3주면 전 무기 자동화 가능**
- 무기마다 3문제: 첫 문제 막히면 → 위의 무기 정의 다시 보고 → 다음 문제로
