## ① 진위 추리 (Knight/Knave) (4문제)

### 문제 패턴
'진실/거짓' 섞인 진술 분석.

### 핵심 무기
- **가정·검증**: 한 사람을 진실/거짓 가정 → 모순 확인
- **경우 분석표**: 가능한 모든 진위 조합 표로
- **메타 진술**: '우리 중 N명 이상' → N에 대한 경우 분석
- **역추적**: 결과에서 거꾸로 따라가 시작 상태 찾기

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `01. NM 2008/12.png` | Andy/Ben/Calvin 줄 서기 총 인원 추리 |
| `31. Mock/5_21.png` | 4명 진술 중 정확히 1명만 옳을 때 그림 수 구하기 |
| `31. Mock/5_34.png` | 5명 진술 중 3명만 참일 때 3자리 수 찾기 |
| `31. Mock/4_22.png` | 6명이 순서대로 절반 빼고 1개 돌려놓는 사탕 문제 역산 |

💡 **Tip:** '정확히 N명이 참' 조건 = N을 1,2,...로 다 시도.

## ② 마방진 (2문제)

### 문제 패턴
n×n 칸의 각 줄·열·대각선 합이 같음.

### 핵심 무기
- **3×3**: 중앙 = 전체 합/9. 한 줄 합 = 전체 합/3
- **중앙 셀**: 대각선·중앙줄에 다 포함 (4회) → 결정 단서
- **부분 채워진**: 미지 칸 변수로 두고 줄별 식

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/1_29.png` | 3×3 마방진에서 x 구하기 |
| `31. Mock/2_20.png` | 3×3 마방진 일부 채워진 공통합 구하기 |

💡 **Tip:** 3×3 마방진은 중앙 = 평균값이 핵심.

## ③ 스도쿠형 격자 논리 (1문제)

### 문제 패턴
격자에 조건 만족 배치.

### 핵심 무기
- **제거법**: 올 수 없는 값부터 지우기
- **유일성**: '이 행에서 X 들어갈 자리 1곳뿐' 찾기
- **경우 분기**: 가정 → 모순 확인

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `54. RI 2024/01.png` | 5×5에 1~5 배치, 각 영역합 같을 때 '?' 값 |

💡 **Tip:** 가장 제약 많은 칸부터 결정.
