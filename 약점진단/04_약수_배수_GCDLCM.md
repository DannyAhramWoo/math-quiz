## ① 약수와 배수 (기본 + 정수론 일반) (9문제)

### 문제 패턴
약수/배수 관계 활용. 정수론의 기초 도구.

### 핵심 무기
- **약수 판정**: 2→끝자리 짝수, 3→자릿수합 3배수, 5→끝자리 0/5
- **소인수분해**: 모든 자연수 = 소수의 곱 (유일)
- **배수의 성질**: a, b가 n의 배수 → a±b도 n의 배수

## ② GCD/LCM 관계 (4문제)

### 문제 패턴
두 수의 최대공약수, 최소공배수 활용.

### 핵심 무기
- **핵심 공식**: GCD(a,b) × LCM(a,b) = a × b
- **A=ka, B=kb** (k=GCD, gcd(a,b)=1) → LCM = k·a·b
- **A+B = k(a+b), LCM = k·a·b** → 분해 가능
- **유클리드 호제법**: GCD(a,b) = GCD(b, a mod b)

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/2_21.png` | A+B=2024, LCM=3795일 때 큰 값 |
| `51. RI 2021/20.png` | n명 원형 게임, 자기 외 모두 받는 라운드 보장 최소 n |
| `54. RI 2024/09.png` | [1,2,...,n]/n! 의 서로 다른 값의 개수 |
| `56. AP 2021/07.png` | 4개의 서로 다른 자연수 합 1111의 최대 공약수 |

💡 **Tip:** GCD/LCM 둘 다 주어지면 항상 'A=ka, B=kb' 변수화.

## ③ 약수의 개수 & 서로소 (3문제)

### 문제 패턴
소인수분해 결과로 약수 개수 즉시 계산. 서로소 활용.

### 핵심 무기
- **약수 개수**: n=p^a·q^b·r^c → 약수 수 = (a+1)(b+1)(c+1)
- **약수의 합**: σ(n) = (1+p+...+p^a)(1+q+...+q^b)...
- **서로소**: GCD=1, 즉 공통 소인수 없음
- **제곱수의 약수 개수는 홀수**

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `52. RI 2022/11.png` | 라벨 1~1000 캔디 증감 후 20개 넘는 학생 수 |
| `56. AP 2021/13.png` | 2와 7로 나뉘지만 동시 아닌 약수 수 |
| `60. AP 2025/05.png` | 직전 특수수 합과 서로소인 20번째 특수수 |

💡 **Tip:** 약수 9개 = 3×3 = (2+1)(2+1) → n=p²q² 형태.
