## ① 색칠/채색 경우의 수 (4문제)

### 문제 패턴
인접 영역은 서로 다른 색으로 칠하는 문제.

### 핵심 무기
- **기본 곱셈**: 한 칸부터 색 선택 → 이웃은 (전체-기정) 가지
- **chromatic polynomial** P(G,k)
- **삭제·축약 공식**: P(G,k) = P(G-e,k) - P(G/e,k)

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/4_35.png` | 11개 육각형 칸 4색으로 인접 다르게 칠하는 경우의 수 |
| `31. Mock/5_33.png` | 3×5 격자 9칸 검정칠, 각 행·열 홀수 검정 경우의 수 |
| `52. RI 2022/17.png` | 3×3 격자 홀짝 인접 배치 경우의 수 |
| `60. AP 2025/20.png` | 8개 영역 인접 다르게 4색 칠하기 수 |

💡 **Tip:** '~색으로 칠하기, 인접 다르게' = 그래프 + 한 점씩 결정.

## ② 정수 분할 (별과 막대) (4문제)

### 문제 패턴
n을 k개 정수 합으로 나누기.

### 핵심 무기
- **음이 아닌**: C(n+k-1, k-1)
- **양의 정수**: C(n-1, k-1)
- **제약 조건**: 최소값·최대값 있으면 치환으로 단순화
- **엄격 증가**: 정확히 다른 값으로 분할

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/5_23.png` | 30개를 4묶음에 n번째에 최소 n개 조건으로 분배 |
| `53. RI 2023/17.png` | 사과 23, 오렌지 18을 5상점에 최소 조건 만족 분배 수 |
| `31. Mock/1_14.png` | 구슬 28개를 3상자에 빈 상자 없이 분배 |
| `56. AP 2021/12.png` | 6일간 매일 증가하며 25문제 푸는 분할 수 |

💡 **Tip:** '사탕 N개를 k명에게' = 거의 다 별-막대.

## ③ 비둘기집 원리 (2문제)

### 문제 패턴
n+1을 n에 → 한 곳에 2개. 다양한 응용.

### 핵심 무기
- **기본**: n+1 항목 → n 그룹 → 한 그룹 최소 2개
- **강화**: kn+1 항목 → n 그룹 → 최소 k+1개
- **경우 수 활용**: 가능한 결과 N가지 → N+1명이면 같은 결과 2명

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/4_32.png` | 1~13 카드에서 두 카드 쌍합이 모두 다른 최대 카드 수 |
| `31. Mock/5_30.png` | 5색 구슬 2개씩 뽑을 때 같은 조합 2명 보장 최소 인원 |

💡 **Tip:** '항상 ~한 사람이 있다' = 비둘기집 신호.

## ④ 격자 경로 (2문제)

### 문제 패턴
(0,0)→(m,n) 오른쪽/아래로만 이동.

### 핵심 무기
- **기본**: C(m+n, m)
- **경유점 P 통과**: (시작→P) × (P→끝)
- **Y/P 둘 다 통과**: 곱셈으로 분해

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `51. RI 2021/18.png` | 8-0에서 11-8까지 A 승리로 가는 경로 수 |
| `56. AP 2021/22.png` | 원형 트랙 두 사람 만남으로 둘레 구하기 |

💡 **Tip:** '최단 경로 + 경유점' = 두 단계로 곱하기.
