## ② 모듈러 산술 (합동·주기) (3문제)

### 문제 패턴
큰 수 직접 계산 없이 나머지로 분석.

### 핵심 무기
- **기본**: (a+b) mod n = ((a mod n)+(b mod n)) mod n. 곱셈도 동일
- **페르마 소정리**: p 소수, gcd(a,p)=1 → a^(p-1) ≡ 1 (mod p)
- **오일러 정리**: gcd(a,n)=1 → a^φ(n) ≡ 1 (mod n)
- **중국인의 나머지 정리 (CRT)**: 서로소 모듈러 합치기
- **주기**: a^k mod n 의 주기는 짧음 (예: 7^k mod 100 주기 4)

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/4_33.png` | 2^2020, 3^2020을 100으로 나눈 나머지 차이 계산 |
| `55. RMO 2025/04.png` | BG=EF, 90도 교차 조건에서 ∠CEF 구하기 |
| `31. Mock/2_32.png` | (2^1+1)…(2^2011+1) 마지막 두 자리 |

💡 **Tip:** 큰 거듭제곱 mod 100 = 'CRT (mod 4 × mod 25)' 시도.

## ③ 팩토리얼 분석 (2문제)

### 문제 패턴
n!의 소인수, 끝자리 0의 개수.

### 핵심 무기
- **Legendre 공식**: n!에서 소수 p의 지수 = [n/p]+[n/p²]+[n/p³]+...
- **끝자리 0** = n!에서 5의 지수 (보통 5가 2보다 적음)
- **예**: 26! 끝자리 0 = [26/5]+[26/25] = 5+1 = 6개

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/2_33.png` | 26! 표기에서 미지수 A+B+C+D 합 |
| `55. RMO 2025/15.png` | 2025!의 끝에 붙는 0의 개수 |

💡 **Tip:** 끝자리 0 = 5의 지수 세기.

## ④ 곱셈 마방진 (1문제)

### 문제 패턴
각 줄·열의 곱이 같은 마방진. log 변환으로 합 마방진.

### 핵심 무기
- **log 변환**: 곱 ↔ 합으로 바꿈 (3×3 합 마방진의 중앙=평균값)
