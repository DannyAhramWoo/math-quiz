# 📐 각도·닮음·원 — 평면기하 핵심

평면 기하의 가장 큰 두 축은 **각도 추적**과 **닮음/비례**. 원의 성질까지 익히면 평면기하 문제의 80%는 해결.


**이 단원의 총 문제 수: 21개**

## ① 각도 추적 (외각·내각·평행선) (4문제)

### 문제 패턴
주어진 도형 안 여러 각도를 차례로 추적해서 미지각을 구하는 문제.

### 핵심 무기
- **외각 정리**: 삼각형 외각 = 인접하지 않은 두 내각의 합
- **내각 합**: 삼각형 180°, n각형 (n-2)×180°
- **평행선 동위각·엇각** = 같음
- **이등변삼각형**: 두 밑각 = 같음

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/4_18.png` | 9시 지난 후 시침·분침이 12시 방향에 대칭이 되는 시각 |
| `31. Mock/4_24.png` | 정사각형 6개로 만든 직사각형에서 각 ABC 구하기 |
| `52. RI 2022/07.png` | 두 평행사변형과 AC=AD=CF 조건에서 ∠ECD 구하기 |
| `54. RI 2024/20.png` | 별 도형에서 ∠C+D+E+F+G 합 구하기 |

💡 **Tip:** 복잡한 그림은 한 각도씩 표시하며 추적. 종이에 모든 각 라벨 적기.

## ② 닮음 & 삼각형·사각형 성질 (8문제)

### 문제 패턴
닮음 조건 확인 후 변의 비/넓이 비 계산. 사각형의 특수 성질 활용.

### 핵심 무기
- **닮음 조건**: AA (각 2개 같음) — 가장 자주 사용
- **닮음비 k → 둘레비 k → 넓이비 k²**
- **중점연결정리**: 두 변의 중점 연결선 = 나머지 변의 1/2, 평행
- **평행사변형**: 마주보는 변 = 같음, 대각선이 서로 이등분
- **사다리꼴 넓이** = (윗변+아랫변)×높이÷2

### 해당 문제

|---|---|
| `52. RI 2022/02.png` | 평행선 분할 삼각형의 평행사변형 넓이로 ABC 넓이 구하기 |
| `31. Mock/1_10.png` | 정육각형·정오각형 결합 시 각 BAC |
| `31. Mock/2_14.png` | 팔각형 그림 속 삼각형 총 개수 |
| `53. RI 2023/14.png` | 겹치는 넓이 578로 AF 길이 구하기 |
| `60. AP 2025/28.png` | 정12각형 3점 선택 시 둔각삼각형 개수 |
| `31. Mock/4_34.png` | BE+BF=49 조건에서 삼각형 DEF 최소 넓이 |
| `31. Mock/5_14.png` | 계단 모양 절단 후 정사각형 만들 때 AB:BC 비율 |
| `31. Mock/2_15.png` | 작은 정사각형들로 분할된 직사각형 둘레 |

💡 **Tip:** 닮음 찾으면 항상 '어떤 두 변이 대응?' 묻고 비율 적기.

## ③ 원의 성질 (5문제)

### 문제 패턴
원과 직선·각이 만드는 도형의 길이/각도.

### 핵심 무기
- **원주각 정리**: 같은 호의 원주각 = 모두 같음. 중심각 = 원주각의 2배
- **탈레스**: 반원에 내접한 각 = 90°
- **접선·반지름**: 접점에서 반지름과 접선은 직각
- **내접사각형**: 마주보는 두 각 합 = 180°

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `52. RI 2022/14.png` | 2022/(1+2+..+k) 합의 자릿수 합 |
| `52. RI 2022/19.png` | 원형 트랙 두 자전거 같은/반대 방향 만남으로 둘레 구하기 |
| `53. RI 2023/08.png` | 정사각형 내부를 따라 원이 회전할 때 덮는 넓이 |
| `56. AP 2021/10.png` | 7개 단위정사각형 도형의 외접원 넓이 |
| `56. AP 2021/24.png` | 수직 현으로 나뉜 4영역 넓이 차 |

💡 **Tip:** 원 안에 점/선 있으면 '중심을 잇는 보조선' 그리는 습관.

## ④ 피타고라스 & 넓이 분할 (4문제)

### 문제 패턴
직각삼각형 세 변 관계 + 복잡 도형을 잘라서 넓이 계산.

### 핵심 무기
- **a² + b² = c²** (c=빗변)
- **자주 나오는 세트**: 3-4-5, 5-12-13, 8-15-17, 7-24-25
- **넓이 분할**: 복잡 도형을 삼각형·직사각형으로 나눠 더하거나 빼기

### 해당 문제
| 파일 | 한줄 요약 |
|---|---|
| `31. Mock/1_27.png` | 직사각형 종이 접어 A=C일 때 EF 길이 |
| `31. Mock/1_16.png` | 직사각형 내 삼각형 두 넓이로 ADOE 구하기 |
| `53. RI 2023/15.png` | 변 3,5,8 정사각형 3개에서 대각선 만든 빗금 넓이 |
| `60. AP 2025/06.png` | 정사각형 안 두 음영 삼각형 합 넓이 |

💡 **Tip:** 직각이 있으면 무조건 피타고라스 떠올리기.
