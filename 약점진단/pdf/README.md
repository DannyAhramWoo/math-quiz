## 📄 각 PDF 구성

각 단원의 PDF는 다음 순서:

### 1부. 핵심 무기 연습 (3문제 × N무기)
- 하위 그룹별 묶음
- 각 무기 헤더 + 정의 박스
- 3문제 + 답(주황색) + 풀이(회색 작은 글씨)

### (새 페이지)

### 2부. 개념 종합 연습 (5문제 × N개념)
- 세부개념별 묶음
- 5문제 + 답 + 풀이

## 📝 인쇄 추천 설정

- **용지**: A4
- **방향**: 세로
- **양면 인쇄**: ON (종이 절약)
- **여백**: 기본 (2cm 좌우, 1.5cm 상하)
- **흑백/컬러**: 컬러 추천 (답·풀이 색상 구분)
- **양면일 때**: '단원별로 인쇄 후 묶기' 추천 (한 단원당 평균 10-20페이지)

## 🎯 추천 학습 순서

1. **우선순위 단원**부터 출력 (약점진단 결과)
   - 🔴 01 (비/비율, 19문제 약점)
   - 🔴 05 (자릿수/진법, 10문제 약점)
   - 🔴 07 (포함-배제, 9문제 약점)

2. **각 단원 안에서**:
   - 무기 연습부터 (자동화)
   - 개념 연습 (종합 응용)

3. **반복**: 틀린 문제는 일주일 뒤 다시
